# Student Manager Python (V1)

CLI based Student Management System using Python file handling.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Data stored in TXT file

## Technologies Used

- Python
- File Handling
- CLI (Command Line Interface)

## Project Structure

student-manager-python/
│
├── main.py
├── data.txt
├── README.md

## How to Run

1. Clone repository

https://github.com/shashankA137/student-manager-python.git

2. Navigate to project

cd student-manager-python

3. Run program

python main.py
